Docker Manager - Online Docker Client API

Docker manager is an online way to connect and manage your docker server. Docker Manager is able to list all of your containers whether it is running or not. The containers can be started, stopped by a single click on your mouse. You will be informed about your containers in seconds.

Usage:

  • Connection:
    • Go to Connect page by the menu.
    • In the text field, write in your docker server's IP address and port separated by a colon.
    • Click on Connect button which will connect your Docker server automatically.
  • Manage your containers:
    • Go to Containers page by the menu. The Docker Manager lists all of your containers in a selectable list.
    • Select the one container by clicking on it's image name.
    • With the action buttons you can start actions on your selected container.
  • Create new container:
    • Go to Create page in the menu.
    • Fill all the data in the three textboxes.
    • By clicking on Create button, your can create the new container.
