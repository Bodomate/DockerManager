Docker Manager - Online Docker Client API

Docker manager is an online way to connect and manage your docker server. Docker Manager is able to list all of your containers whether it is running or not. The containers can be started, stopped by a single click on your mouse. You will be informed about your containers in seconds.

Usage:

  • Connection: Please go to Connect page by the menu. In the text field, write in your docker server's IP address and port separated with a colon. Then click on Connect button which will connect your docker automatically.
  • Reach your containers: Go to Containers page by the menu. The Docker Manager lists all of your containers in a selectable list. Select the one container by clicking on it's image name. With Start and Stop button you can run and stop your selected container. Click on Info button to list additional information of the container.
